# Tachyon – A College Fest Website

Tachyon is a modern, responsive, and interactive website created to represent a college fest. It includes pages for events, teams, sponsors, and contact, all designed to provide an engaging experience for students and visitors.

---

## Overview

This project aims to create a digital platform that highlights the essence of a college festival.  
Users can explore event details, meet the organizing team, and learn more about the fest’s sponsors and schedule.

---

## Features

- **Home Page:** Introduction and overview of the fest  
- **Events Page:** List of all fest events with details  
- **Team Page:** Core team members and their roles  
- **Sponsors Section:** Display of event sponsors  
- **Contact Page:** Contact form and social media links  
- **Responsive Design:** Works smoothly on all devices
